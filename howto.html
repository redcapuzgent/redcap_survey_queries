How to use the imagemapcheckboxes hook:

Prerequisites
  • To use this hook you need to know about images and imagemaps.
  • You will need an imagemap. You can use the following tool to do this: https://www.image-map.net
Using the hook
  • Create a field with the required checkboxes. In this example called 'checkboxez'.
  • Create a descriptive text field. Add the image tag and the area map as the field label. eg. 
    <img  id="colorme" src="http://127.0.0.1/redcap/redcap/hooks/server/global/imagemapcheckboxes/painmap_female.png" usemap="#mymap">
